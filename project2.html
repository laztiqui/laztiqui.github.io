Machine Learning Project

With the rise in machine learning and artificial intelligence, this project gave me exposure to the process of using machine learning, as well as updating code that is already out there. This project demonstrates my proficiency in Python and Arduino programming languages, while also using a real-life example.

In the first part of this project, we were given two machine learning algorithm. All I had to do was give it some data to train on. The first program could reconize face structure and map filters on top of faces. The second program was trained to recognize faces. The training data included a few celebrities, and I also gave it data to recognize my own face.

In the last part of this project, we were tasked with building a face tracking camera stand. We were given two servo motors and a stand, which he had to put together ourselves. We also were given partially correct code that would track our face and adjust the position of the stand so that our face was always in the middle of the screen. We also had to wire the schematic and make sure all the connections were correct. Our task was to fix the code and successfully implement this face tracker. See video below.
